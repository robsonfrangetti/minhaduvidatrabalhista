║                    ARQUIVOS MAIS IMPORTANTES                           ║
╚════════════════════════════════════════════════════════════════════════╝

1️⃣  data/topics.ts
    → Contém TODOS os dados dos temas
    → EDITE AQUI para adicionar/modificar temas
    → Estrutura: legislação, portarias, súmulas, etc.

    → Página inicial com cards de temas
    → Lista todos os temas disponíveis

3️⃣  app/resultado/[slug]/page.tsx
    → Exibe resultados completos
    → Busca informações, súmulas, decisões

4️⃣  app/globals.css
    → Todos os estilos do site
    → Edite para personalizar cores e layout

5️⃣  README.md
    → Documentação principal
    → Instruções de instalação e uso

╔════════════════════════════════════════════════════════════════════════╗
║                    COMO ADICIONAR NOVO TEMA                            ║
╚════════════════════════════════════════════════════════════════════════╝

1. Abra: data/topics.ts

2. Adicione um novo objeto seguindo o padrão:

   "horas-extras": {
     id: "horas-extras",
     title: "Horas Extras",
     description: "Informações sobre horas extras",
     keywords: ["horas extras", "50%", "100%"],
     legislation: [...],
     ordinances: [...],
     summaries: [...],
     jurisprudence: [...],
     recentDecisions: [...]
   }

3. Salve o arquivo

4. Teste: http://localhost:3000

5. Pronto! O novo tema aparecerá automaticamente!
